- [React](https://reactjs.org/) – Componentes y hooks  
- [Next/Image](https://nextjs.org/docs/api-reference/next/image) – Optimización de imágenes

## 🚀 Instalación y desarrollo local

Clona el repositorio y ejecuta el proyecto localmente:

```bash
# Clonar el repositorio
git clone https://github.com/tu-usuario/portafolio-erick-hernandez.git
cd portafolio-erick-hernandez

# Instalar dependencias
npm install
# o
yarn install
# o
pnpm install

# Ejecutar en modo desarrollo
npm run dev
